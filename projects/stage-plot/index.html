Stage Plot Tool

An interesting tool I made so I could learn Fabric.js and Javascript in general. In Flashband we were using Google Draw documents for our one-night bands to make stage plots. This is to replace that with a dedicated tool.

Still a work in progress. When I finish it up it will go on the Flashband website. The 'Generate Output' and 'Load Input' boxes are to demo the save / reload functionality that will link to our database.

Make a stage plot

Add a musician, update their name, then link amps and microphones to them.
